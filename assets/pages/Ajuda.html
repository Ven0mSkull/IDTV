Desafio 1 - SQL Injection

Este desafio testa sua capacidade de explorar vulnerabilidades de injeção SQL. O objetivo é manipular uma consulta SQL para acessar dados não autorizados.

  • Dica 1: Tente usar ' OR '1'='1 no campo de login.
  • Dica 2: Observe como o sistema responde a entradas inválidas.
  • Solução: Use admin' -- para ignorar a verificação de senha.

Desafio 2 - XSS Básico

Aqui você deve injetar um script malicioso em um campo de entrada para executar código no navegador.

  • Dica 1: Experimente <script>alert('XSS')</script>.
  • Dica 2: Veja se o sistema filtra tags HTML.
  • Solução: Use um script simples que mostre uma mensagem.

Desafio 3 - Brute Force

O objetivo é descobrir uma senha testando várias combinações.

  • Dica 1: Use uma lista de senhas comuns (ex.: "123456").
  • Dica 2: Automatize com um script simples se permitido.
  • Solução: A senha é "password123".
